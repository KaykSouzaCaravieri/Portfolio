Sobre Mim

Olá! Sou desenvolvedor focado em criar soluções simples, modernas e funcionais para negócios e projetos pessoais.

Trabalho com HTML, CSS e JavaScript, desenvolvendo landing pages, sites institucionais e pequenas automações, sempre com foco em:

-Design limpo e moderno.

-Boa experiência para o usuário.

-Código organizado e fácil de manter.

Meu objetivo é transformar ideias em páginas que realmente convertem, transmitam confiança e tragam resultados reais. Por isso estou sempre estudando e evoluindo, buscando entregar cada projeto com qualidade, rapidez e atenção aos detalhes.

Loja de Carros

Site de loja de carros com foco em visual moderno, navegação intuitiva e experiência do usuário. O projeto apresenta catálogo de veículos, layout responsivo e interface organizada. O objetivo foi criar uma plataforma clara e atrativa para apresentar os veículos, facilitando a visualização e o contato com o vendedor.

Hamburgueria

Sistema para uma Hamburgueria que permite ao cliente se cadastrar, fazer login e realizar pedidos de forma rápida e simples. O sistema organiza os pedidos, gera um resumo automático e permite a impressão para facilitar o atendimento no balcão.

Jogo de Cartas

Desenvolvi um jogo de cartas interativo inspirado em jogos de duelo, criado com HTML, CSS e JavaScript. O projeto apresenta um sistema de navegação entre cartas, animação de virar a carta ao clicar, design temático e interface responsiva.
